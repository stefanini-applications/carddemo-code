📦 Core Modules

🔐

Authentication

User login/logout and session security. COSGN00C + USRSEC VSAM.

CC00 · COSGN00C
🏦

Account Management

View and update credit card accounts: balances, limits, status.

CAVW/CAUP · ACCTDATA VSAM
💳

Card Management

List, view, and update credit cards. Encrypted card numbers + CARDAIX.

CCLI/CCDL/CCUP · CARDDATA VSAM
💰

Transaction Management

List, view, and add credit card transactions. Online real-time entry.

CT00/CT01/CT02 · TRANSACT VSAM
📊

Reports

Aggregated transaction statistics by category from TCATBALF.

CR00 · CORPT00C
💸

Bill Payment

Process customer payments and apply to account balances.

CB00 · COBIL00C
👥

User Management

Admin-only: add, update, delete system users with role types.

CU00–CU03 · USRSEC VSAM
⚙️

Batch Processing

Nightly cycle: post transactions, calculate interest, generate statements.

POSTTRAN · INTCALC · CREASTMT

🔌 Optional Extension Modules

🛡️

Authorization (IMS+DB2+MQ)

CP00/CPVS/CPVD · MQ · IMS · DB2
🏷️

Transaction Types (DB2)

Manage DB2-backed transaction type catalog via CICS online screens.

CTLI/CTTU · DB2 TRANSACTION_TYPE
