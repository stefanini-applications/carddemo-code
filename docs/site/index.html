📦 Core Modules

🔐 Authentication

User signon, credential validation, and role-based menu routing via USRSEC VSAM file.

Transaction: CC00

🏦 Accounts

View and update credit card account information including balances, limits, and dates.

Transactions: CAVW, CAUP

💳 Credit Cards

List, view, and update credit card records linked to accounts via cross-reference.

Transactions: CCLI, CCDL, CCUP

💸 Transactions

List, view, and add credit card transactions against the TRANSACT VSAM KSDS.

Transactions: CT00, CT01, CT02

💰 Bill Payment

Process online bill payments — pay account balance in full with transaction recording.

Transaction: CB00

📊 Reports

Generate transaction reports and account statements via batch job submission from CICS.

Transaction: CR00

👥 User Management

Admin CRUD operations on the USRSEC user security file.

Transactions: CU00–CU03

⚙️ Batch Processing

Nightly transaction posting, interest calculation, data loading, and reporting jobs.

JCL / COBOL Batch
