📚 Core Modules

🔐 Authentication

authentication

Signon and session management. RACF/VSAM credential validation, user type routing (regular vs admin), CICS COMMAREA session state.

CC00 · COSGN00C · USRSEC

🏦 Accounts

accounts

View and update credit card account information: balances, credit limits, dates, and group assignments for interest calculation.

CAVW · CAUP · COACTVWC · ACCTFILE

💳 Credit Cards

credit-cards

List, view, and update credit cards linked to accounts. Manages card status, CVV, expiration dates, and account cross-references.

CCLI · CCDL · CCUP · CARDFILE

💸 Transactions

transactions

Browse, view, and add credit card transactions. Includes online CICS screens and batch posting via POSTTRAN for daily transaction processing.

CT00-CT02 · CBTRN02C · TRANSACT

🧾 Bill Payment

bill-payment

Process bill payments to reduce account balances. Payments recorded as transaction credits and validated against account status.

CB00 · COBIL00C

📈 Reports

reports

Generate transaction reports and statements. Online trigger from CICS submits batch job via Internal Reader; CBTRN03C prints detail reports.

CR00 · CORPT00C · CBTRN03C · TRANREPT

👥 User Management

user-management

Admin-only CRUD for CardDemo users. Manages user credentials, names, and type (Regular/Admin) in the USRSEC VSAM file.

CU00-CU03 · COUSR00C-03C · USRSEC

⚙️ Batch Processing

batch-processing

Full nightly batch suite: file loading, transaction posting, interest calculation, statement generation, backup, and VSAM index rebuild.

POSTTRAN · INTCALC · COMBTRAN · JCL

🔧 Optional Extension Modules

🛡️ Authorization

authorization

Real-time credit card authorization via MQ. Integrates IMS DB for hierarchical storage, DB2 for fraud analytics, and MQ for request/response messaging.

CP00 · CPVS · CBPAUP0C · IMS · DB2 · MQ

🗄️ Transaction Type DB2

transaction-type-db2

Manage transaction type reference data in DB2 tables. Demonstrates static embedded SQL, cursor processing, and DB2 CRUD patterns in CICS.

CTTU · CTLI · COTRTUPC · DB2

📨 MQ Integration

mq-integration

Asynchronous MQ request/response patterns for system date and account detail inquiries. Demonstrates decoupled integration with CICS and VSAM.

CDRD · CDRA · CODATE01 · MQ
