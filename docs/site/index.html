📚 Core Modules
🔐 Online

Authentication

CICS signon (CC00) with RACF-backed VSAM user security file. Routes users to Regular or Admin menus.

🏦 Online

Account Management

View (CAVW) and update (CAUP) credit account records — balance, credit limit, status, and group assignment.

💳 Online

Card Management

List (CCLI), view (CCDL), and update (CCUP) credit cards. Manages card status, embossed name, and expiration.

💰 Online + Batch

Transaction Processing

Online transaction list/view/add (CT00–CT02) and batch daily posting (POSTTRAN/CBTRN01C) to account balances.

💵 Online

Bill Payment

Full-balance bill payment (CB00). Posts a payment transaction and reduces the account balance in ACCTDAT.

📊 Reporting

Reporting

Online report request (CR00) that submits batch JCL via TDQ. Statement generation (CBSTM03A/B) and data export.

👥 Admin

User Management

Admin-only CRUD for application users (CU00–CU03). Manages USRSEC VSAM file for authentication.

⚙️ Batch

Batch Processing

Full nightly batch cycle: file init, transaction posting (POSTTRAN), interest calculation (INTCALC), statement generation, file open/close.

🔌 Optional Extension Modules
🔒 IMS/DB2/MQ

Authorization IMS/DB2/MQ

Credit card authorization via MQ messaging, IMS DB for pending auth records, Db2 for transaction logging. Includes purge batch job.

🗄️ DB2

Transaction Type DB2

Admin management of transaction types and categories stored in Db2 (CTTU/CTLI). Demonstrates cursor, insert, update, delete on Db2.

📨 MQ

VSAM/MQ Integration

MQ request/response patterns for account inquiry (CDRA) and system date inquiry (CDRD) — enables asynchronous external access.
